Bu proje 90Pixel tarafından iletilen front-end code challange için, ReactJS ve ReduxJS kullanılarak yapılmıştır.
Geliştirme ortamı olarak VS Code kullanılmıştır.

Amaç:
1-) omdbapi kullanarak imdb'de yer alan film ve dizileri search box ile arama,
2-) arama sonucu bulunan film ve dizilerin poster,ad ve yıl bilgilerini getirme,
3-) detay butonu sayesinde film veya dizinin detay bilgilerini görme

Kullanılan komutlar:
1-) npx create-react-app imdb-app
2-) npm install --save redux
3-) npm install --save redux-devtools-extension
4-) npm install --save redux-thunk
5-) npm install --save axios
6-) npm install --save react-router-dom
